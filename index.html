Course Introduction

Course Aims

This course aims to equip students with the fundamental principles and techniques to design and create virtual world which are suitable for interactive applications. Virtual world is a self-contained, persistence computer simulated environment in which player can inhibit, navigate and interact with non-player objects. Student will learn the essential techniques and skills to create an engaging and dynamic virtual environment with physical simulations, intelligent responses and real-time interactions.

Course Intended Learning Outcomes (CILOs)

Upon successful completion of this course, students should be able to:

  • 1. Master the basic principles and theories of virtual world design and creation;
  • 2. Develop immerse interaction through various input;
  • 3. Integrate realistic and physical-based animation and dynamic effects in virtual environment and interface designs;
  • 4. Integrate and synthesis creative applications based on the knowledge of virtual world design.
*from HKU-Dreamscape-Design-and-Interaction-Course-Outline-Project-Outline-2020-Schedule issued by HKUSI

Program Introduction

Overview

AAH-I is an NGO that provides the refugees with skill training on handicrafts and clothing manufacturing. This summer, China House cooperates with Action Africa Help International (AAH-I) to build a shopping site for refugees to promote and sell their products. Students will learn programming foundations of front end and back end to build their portfolio websites and then help to construct the shopping site.

Aim:

The aim of the programme is to deliver the concept of refugee’s real needs and help them through an efficient way by building the shopping website.

Objectives:
  • 1. Understand the meaning of the programme
  • 2. Communicate with clients and design UX/UI mockups.
  • 3. Learn programming foundations of HTML, CSS, and JavaScript.
  • 4. Utilize Bootstrap to create a simple website.
  • 5. Build own portfolio site.
  • 6. Publish the website.
  • 7. Understand how the backend works
  • 8. Help to construct a shopping site for an NGO organization
*from 中南屋 2019 难民营手工艺品网站开发项目课程大纲(China House 2019 Syllabus of the refugee Camp Handicraft Website Development Project) issued by China House
